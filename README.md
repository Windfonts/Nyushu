# 𛆁𛈬𛇋𛆼｜女书柳叶（暂名）｜Nüshu “Serif”
一套女书｢柳叶体(衬线体)｣书法字体，旨在通过OT特性支持女书各种异体，允许同一女字变换各种写法。

未来计划：字重可变、衬线向无衬线可变，实际上绘制曲线的同时已经为可变字体作准备。

# [NyushuSrc.csv]
- 汉字仅第一个依 徐焰《女书规范字书法字帖》，多于一字表示本字体加了异体字形（或有误），括号内的为一形对多字（不一定准确）
- `≈`标的汉字是统一码NamesList.txt备注的，其中｢抆｣有勘误，应为｢坟｣。与《字帖》不同的以`!`号标记。
- 经过勘误的备注`up!`
- 读音（kReading）以 [NushuSources.txt](https://www.unicode.org/Public/15.0.0/ucd/NushuSources.txt) 为准。
- “偏旁”为自撰，方便我字体制作，不一定准确，未完。其中`。`表示其偏旁即其本身，` \ `是表示此字形多种拆法的分界。偏旁排序未按先中间后两边的笔顺来。

女书对应的是江永土话音节，因此其数量有上限，不可无限加字，但可有异体。已知有些土话音节底层不是汉语，因此女字不能与汉字一一对应。

# 女书排版
原本不用标点或空格，某种程度上算音节文字，但大多女字是多音字（一对多。而且也有同音异形字，多对一）所以不是。实际上女书在同一篇文中对同一事物会用不重复的女字表达，避免文章呆板[^NHZD]。上面提到的𛈺｢伴｣，女漢字典称有40多个变体。

女书叠字号｢𖿡｣是必须的，《女漢字典》收录变体超过6种，并非图省笔的略写，即便是｢人人｣也得写作｢𛅳𖿡｣。
> 例如[女书提案09-03-18](http://std.dkuug.dk/jtc1/sc2/wg2/docs/n3598.pdf)，64页

女书“换行符”是可见的，有2个，称为行尾填空符，形如↓、o，填补行列的剩余空间，统一码截至15.0版未收录。我需要斟酌字体该怎么做。
